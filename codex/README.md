# Codex Artifact

This directory contains the Dropbox Codex provider artifact.

The artifact includes `.codex-plugin/plugin.json`, Codex Skills, provider examples, and Codex-only supporting material. Its MCP configuration points at the Dropbox ChatGPT app MCP endpoint used by the Codex plugin.

Keep provider-specific content in this directory, and put only safe cross-provider reference material in `../shared/`.

Before provider submission, complete Product, Content Design, Legal, Security, Analytics, and Engineering review, then record the submitted artifact path, version tag, and commit SHA under `../releases/`.
